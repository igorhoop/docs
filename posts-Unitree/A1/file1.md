# Технические характеристики

Модель A1 производится в **двух** модификациях:
- Basic
- Explorer




|   Параметр  | Basic    | Explorer  |
|------|:-------:|:--------:|
|   Максимальная скорость (м/с)   | 3.3 | 3.3 |
|   Максимальная грузоподъемность (кг) | 5 | 5 |
|   Максимальный угол подъема (°) | 35 | 35 |
|   Максимальная высота ступеней (мм) | 120 | 120 |
|   Бортовые компьютеры | Raspberry Pi 4 / Aaeon UP Developer Board | NVIDIA Jetson Xavier NX / Aaeon UP Developer Board |
|   Камера глубины | Intel RealSense D435 | Intel RealSense D435 |
|   Интерфейсы связи | Wi-Fi, Bluetooth, 2 x HDMI, 2 x Ethernet,2 x USB 3.0, 2 x USB 2.0, 2 отладочных порта | Wi-Fi, Bluetooth, 2 x HDMI, 2 x Ethernet,2 x USB 3.0, 2 x USB 2.0, 2 отладочных порта
|   Датчик Lidar | **-** | **Установлен** |
|   Время работы | 1 час |    1 час       |
